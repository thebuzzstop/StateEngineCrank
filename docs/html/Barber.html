Stopping_Cut()[source]¶

Do function processing for the Stopping state

State machine do function processing for the Stopping state. This function is called once every state machine iteration to perform processing for the Stopping state.

Basically we are performing the same function as the Cutting_Cut function since we are still cutting a customers hair. The main difference is that when we are in this state and the EvFinishCutting event is thrown we will transition to the Finish state.
